An example using non-semantic mark-up

This is one of two sample pages created to demonstrate the diference between pages using semantic HTML mark-up and those that don`t.

What you see isn`t always what you get.

If you compare this page with the one using semantic mark-up, you won`t see significant differences in the browser.

But what`s on the inside can make a world of difference for accessibility, search engine optimization, and workflow. Read the accompnying blog entry to learn more.
